This page is a reference for the various mod settings of Sim Snatcher.

## How to open SS Settings:
- Click on a Sim and go through these interactions -> Sim Snatcher -> Sim Snatcher Settings
- If using the [Mod Settings Menu](https://www.nexusmods.com/thesims4/mods/350) mod, Sim Snatcher Settings will instead appear within the MSM menu and the above interaction will not be available!

# Abduction:
## General Settings:
| Setting Name | What It Does | Default State |
| ------------ | ------------ | ------------- |
| Enable Abduction Interactions | If enabled, the mod will show the abduction interactions in various places. | Enabled |

## Cheat Settings:
| Setting Name | What It Does | Default State |
| ------------ | ------------ | ------------- |
| Abduction Attempts Always Succeed | If enabled, when your Sim attempts to abduct another Sim, they will never fail to abduct them. | Disabled |

# Slavery
## General Settings:
| Setting Name | What It Does | Default State |
| ------------ | ------------ | ------------- |
| Enable Slavery Interactions | If enabled, the mod will show the slavery interactions in various places. | Enabled |

# Order To
## Cheat Settings:
| Setting Name | What It Does | Default State |
| ------------ | ------------ | ------------- |
| Show Debug Interactions In Perform Interaction Dialog | If enabled, when ordering a Captive Sim to do an interaction, debug interactions will be shown within the list of interactions that can be ordered. | Disabled |
